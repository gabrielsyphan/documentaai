# dev

Inicia o DocumentaAI em modo de desenvolvimento.

```bash
npm run tauri dev
```

Se o Rust/Tauri ainda não estiver configurado, inicia só o frontend:

```bash
npm run dev
```

Verifique se todas as dependências estão instaladas (`npm install`) antes de rodar. Se `src-tauri/` não existir ainda, avise o usuário que o projeto precisa ser inicializado primeiro com `/setup`.
